Personal wiki help Markdown

This wiki uses Markdown as language for its pages. Please note that Markdown isn't a standard, and there may be differences.

Cheat sheet
Original specs

Some hints:

  • Use #, ##, ### (etc) before a line to make headers.
  • Use two spaces and an enter after a word to cause a line break after this word.
  • To make a bullet-list, start with two newlines, and prefix each item with "- ".
  • For external links, just paste the link (it will be automatically recognized), or use [link text](http://example.com).
  • Images: ![alt-text](http://example.com/some-image.jpg)
Wikiwords

Wikiwords will be automatically recognized if the word uses mixed casing: "MyNewPage" will be recognized as wikiword, but mynewpage won't be. If you want to manually mark a word as wikiword, surround it by two square brackets: [[mynewpage]].

If you want to prevent a word to be recognized as wikiword, prefix it with a ~, like this: ~ChromeCast.

Tags Include a new line at the end of the file, starting with the word "tags:" to tag a page. Separate the tags by spaces. For example: 
                # Page title

                Page content

                Tags: foo bar
After you save this page, the tags will apear as buttons below the page. Clicking the button shows all pages that have this tag. About
